H1Aging index predicts mental health specialist density
▶

Does aging index correlate with psychiatrist density across Latin American countries?

Variables: aging_index, psychiatrists_per_100k
Method: Pearson correlation + scatter plot (n=11 countries × 24 years)
Expected: Positive correlation — aging populations demand more specialists
Dataset: Mental Health Specialists Dataset
import pandas as pd
from scipy import stats

df = pd.read_csv('ncra_master.csv')
corr, p = stats.pearsonr(df['aging_index'].dropna(), df['psychiatrists_per_100k'].dropna())
print(f'r={corr:.3f}, p={p:.4f}')
df <- read.csv('ncra_master.csv')
cor.test(df$aging_index, df$psychiatrists_per_100k, method='pearson')
H2Dependency ratio correlates with NCRA Risk Score over time
▶

Is the NCRA Risk Score driven primarily by dependency ratio changes (2000-2023)?

Variables: dependency_ratio, ncra_risk_score, year
Method: Panel regression (country fixed effects, n=11, T=24)
Expected: High positive coefficient — dependency ratio is the main driver
Dataset: NCRA LatAm Master Dataset
import pandas as pd
import statsmodels.formula.api as smf

df = pd.read_csv('ncra_master.csv')
model = smf.ols('ncra_risk_score ~ dependency_ratio + C(country_iso)', data=df).fit()
print(model.summary())
df <- read.csv('ncra_master.csv')
library(plm)
pdata <- pdata.frame(df, index=c('country_iso','year'))
model <- plm(ncra_risk_score ~ dependency_ratio, data=pdata, model='within')
summary(model)

H4Demographic dividend index predicts economic productivity
▶

Does the demographic dividend index lead GDP growth by 5-10 years?

Variables: demographic_dividend_index, gdp_per_capita_usd, year
Method: Cross-lagged panel model (lag = 5 years)
Expected: Positive lagged effect — dividend peak precedes productivity gains
Dataset: NCRA Economics Dataset
import pandas as pd

df = pd.read_csv('ncra_master.csv').sort_values(['country_iso','year'])
df['ddi_lag5'] = df.groupby('country_iso')['demographic_dividend_index'].shift(5)
from scipy import stats
corr, p = stats.pearsonr(df['ddi_lag5'].dropna(), df['gdp_per_capita_usd'].dropna())
print(f'Lagged correlation: r={corr:.3f}, p={p:.4f}')
df <- read.csv('ncra_master.csv')
df <- df[order(df$country_iso, df$year),]
df$ddi_lag5 <- ave(df$demographic_dividend_index, df$country_iso, FUN=function(x) c(rep(NA,5),head(x,-5)))
cor.test(df$ddi_lag5, df$gdp_per_capita_usd, use='complete.obs')
H5COVID-19 reversed aging index trends in 2020-2021
▶

Did excess mortality in 2020-2021 temporarily reverse aging index growth?

Variables: aging_index, year, country_iso
Method: Interrupted time series analysis (breakpoint: 2020)
Expected: Temporary dip in aging index 2020-2021, resuming trend by 2022
Dataset: NCRA Temporal Dataset
import pandas as pd
import numpy as np

df = pd.read_csv('ncra_master.csv')
df['post_covid'] = (df['year'] >= 2020).astype(int)
df['time_since_covid'] = np.maximum(0, df['year'] - 2020)
from statsmodels.formula.api import ols
model = ols('aging_index ~ year + post_covid + time_since_covid + C(country_iso)', data=df).fit()
print(model.params[['post_covid','time_since_covid']])
df <- read.csv('ncra_master.csv')
df$post_covid <- as.integer(df$year >= 2020)
df$time_since_covid <- pmax(0, df$year - 2020)
model <- lm(aging_index ~ year + post_covid + time_since_covid + country_iso, data=df)
summary(model)
